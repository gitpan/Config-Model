=head1 NAME

Config::Model::models::LCDd::CwLnx - Configuration class LCDd::CwLnx

=head1 VERSION

version 1.246

=head1 DESCRIPTION

Configuration classes used by L<Config::Model>

=head1 Elements

=head2 Model

Select the LCD model [default: 12232; legal: 12232, 12832, 1602]I<< Optional. Type enum. choice: '12232', '12832', '1602'. upstream_default: '12232'.  >>

=head2 Device

Select the output device to use [default: /dev/lcd]I<< Optional. Type uniline. default: '/dev/ttyUSB0'.  >>

=head2 Size

Select the LCD size [default: depending on model: 12232: 20x4, 12832: 21x4, 1602: 16x2]I<< Optional. Type uniline. upstream_default: 'dependingonmodel'.  >>

=head2 Speed

Set the communication speed [default: 19200; legal: 9600, 19200]I<< Optional. Type enum. choice: '9600', '19200'. upstream_default: '19200'.  >>

=head2 Reboot

Reinitialize the LCD's BIOS [default: no; legal: yes, no]
normally you shouldn't need this. I<< Optional. Type enum. choice: 'yes', 'no'. upstream_default: 'no'.  >>

=head2 Keypad

If you have a keypad connected. Keypad layout is currently not
configureable from the config file.I<< Optional. Type uniline. default: 'yes'.  >>

=head2 keypad_test_mode

If you have a non-standard keypad you can associate any keystrings to keys.
There are 6 input keys in the CwLnx hardware that generate characters
from 'A' to 'F'.

The following is the built-in default mapping hardcoded in the driver.
You can leave those unchanged if you have a standard keypad.
You can change it if you want to report other keystrings or have a non
standard keypad.
KeyMap_A=Up
KeyMap_B=Down
KeyMap_C=Left
KeyMap_D=Right
KeyMap_E=Enter
KeyMap_F=Escape
keypad_test_mode permits to test keypad assignement
Default value is no. I<< Optional. Type uniline. default: 'yes'.  >>
