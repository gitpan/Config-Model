=head2 Type

Set the display type [default: lcd; legal: lcd, lkd, vfd, vkd]I<< Optional. Type enum. choice: 'lcd', 'lkd', 'vfd', 'vkd'. upstream_default: 'lcd'.  >>

=head2 Contrast

Set the initial contrast [default: 480]
NOTE: The driver will ignore this if the display
      is a vfd or vkd as they don't have this feature. I<< Optional. Type uniline. upstream_default: '480'.  >>

=head2 hasAdjustableBacklight

Some old displays do not have an adjustable backlight but only can
switch the backlight on/off. If you experience randomly appearing block
characters, try setting this to false. [default: yes; legal: yes, no]I<< Optional. Type enum. choice: 'yes', 'no'. upstream_default: 'yes'.  >>

=head2 Brightness

Set the initial brightness [default: 1000; legal: 0 - 1000]I<< Optional. Type integer. upstream_default: '1000'.  >>

=head2 OffBrightness

Set the initial off-brightness [default: 0; legal: 0 - 1000]
This value is used when the display is normally
switched off in case LCDd is inactive. I<< Optional. Type integer. upstream_default: '0'.  >>

=head2 Speed

Set the communication speed [default: 19200; legal: 1200, 2400, 9600, 19200]I<< Optional. Type enum. choice: '1200', '2400', '9600', '19200'. upstream_default: '19200'.  >>

=head2 keypad_test_mode

See the [menu] section for an explanation of the key mappings

Afterwards you can modify the settings above and set
keypad_set_mode to no again.I<< Optional. Type uniline. default: 'no'.  >>
